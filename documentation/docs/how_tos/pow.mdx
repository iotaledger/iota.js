# Perform Proof of Work

You can perform Proof of Work(POW) operations using the following functions :

* [NeonPowProvider.pow(message, targetScore)](../references/pow-neon/classes/NeonPowProvider.md#pow).
* [NodePowProvider.pow(message, targetScore)](../references/pow-node/classes/NodePowProvider.md#pow).
* [WasmPowProvider.pow(message, targetScore)](../references/pow-wasm/classes/WasmPowProvider.md#pow).
* [LocalPow.pow(message, targetScore)](../references/client/classes/LocalPowProvider.md#pow).

You will need to provide a `block` and a `targetScore` to any of these functions.  If successful, they will return nonce.
