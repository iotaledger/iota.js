## Troubleshooting

If you are getting the following error, it means your local Proof of Work is taking too long to finish.  By the time it
finishes the parent blocks are no longer valid. You can use [simple connection](simple.mdx) how-to that showcases the
functionalities as this example.

```bash
Error: invalid parameter, error: block is below max depth: invalid block: code=400, message=invalid parameter
```
